# MQL5 Source

This folder contains public-review source files only.

- `USTEC_TrendRunner_v0_4_safety_public.mq5`: educational research EA source sanitized for transparency. It is not approved for live, demo, paper, signal, or managed-account use.
- `USTEC_ExportRatesToJson.mq5`: read-only candle export Script for locally selected symbols and timeframes. It does not trade and does not export account data.

No compiled `.ex5`, preset, tester config, broker report, account statement, market data file, or private optimization artifact belongs in this folder.
