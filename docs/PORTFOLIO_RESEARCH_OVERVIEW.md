# Portfolio Research Overview

EA USTEC Lab is a public technical portfolio project showing how a systematic trading idea can move through implementation, validation, stress testing and rejection.

## What was built

- Educational MQL5 Expert Advisor source for public review.
- A read-only MQL5 rates exporter for local research workflows.
- Python publication guards and tests.
- Public summaries of positive and negative historical backtest evidence.
- Documentation that separates recent positive results from long-horizon robustness failure.

## Technical themes

| Theme | How it appears in the project |
| --- | --- |
| Expert Advisor architecture | MQL5 candidate source and exporter structure |
| Backtest discipline | Strategy Tester results are summarized as research evidence only |
| Risk and drawdown | Maximum drawdown is carried into all public result tables |
| Robustness | 2025 results are compared against 5-year and 10-year windows |
| Negative evidence | Rejected experiments remain documented instead of hidden |
| Publication hygiene | Guards and tests prevent private artifacts and exaggerated claims |

## Portfolio interpretation

The project is not a promoted trading robot. It is a transparent research archive showing that a recent edge can look promising and still fail when the evidence window expands.

That makes it useful as a portfolio artifact: it demonstrates implementation, measurement, review discipline and decision quality under unfavorable evidence.

## Continuation path

The strongest continuation path is regime-first research with explicit no-trade states, early long-horizon checks and fewer strategy changes per experiment.
