# Portfolio Overview

EA USTEC Lab is a technical portfolio archive for systematic trading research on USTEC/Nasdaq CFD. It demonstrates how an idea can move from Expert Advisor implementation to backtest evidence, stress review, long-horizon rejection and public documentation.

## Built artifacts

- Public-review MQL5 research source.
- MQL5 rates exporter.
- Python publication guard and tests.
- Positive and negative backtest summaries.
- Reproducibility, limitation and continuation documents.

## Why it matters

The project is not presented as a trading product. Its portfolio value is evidence discipline: the best 2025 window is visible, and the 5-year and 10-year failures are visible with equal weight.
